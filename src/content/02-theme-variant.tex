\chapter{Theme Variants}

The package currently exposes five themes:

\begin{itemize}
  \item \texttt{purple}
  \item \texttt{black}
  \item \texttt{sepia}
  \item \texttt{ocean}
  \item \texttt{forest}
\end{itemize}

Each theme changes the page palette, cover colors, accent colors, and default
cover treatment.
