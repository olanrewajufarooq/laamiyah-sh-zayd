\chapter{Why Use a Style File}

The point of the package is not to hide LaTeX from you. It is to move repeated
design logic into one place and keep the document entrypoint stable.

\section{What You Can Change}

You can adjust:

\begin{itemize}
  \item theme
  \item trim size
  \item chapter numbering style
  \item cover treatment
  \item running headers
\end{itemize}

\section{What You Still Control Directly}

You still write normal LaTeX content. If you need to extend the design, the
style file is a standard package and can be edited directly.
