\bookfrontmattersection{Preface}
This repository now uses a single XeLaTeX style file rather than a Typst
package. The goal is the familiar LaTeX workflow: choose options at
\verb|\usepackage| time, set metadata once, and keep the document body clean.

\vspace{1.5em}
